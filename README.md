<h3><a id="user-content-description" class="anchor" aria-hidden="true" href="#description"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Description</h3>
<p><a target="_blank" href="/Alain91530/arcade/blob/master/images/arcade.png"><img src="/Alain91530/arcade/raw/master/images/arcade.png" style="max-width:100%;"></a></p>
<p>A classic arcade game following this rules:</p>
<p>The purpose of the game is to have the player cross a road with moving  enemys and reach the river at the top of the game area.<br>
If a collision occurs the player go back to its origin place. At each collision the player lose one life and after 3 lives the game is over and the player lose. The player must do a final move jumping in the river to win.<br>
The player can gain lives by collecting gems on its way to the river. Collecting the yelow gem instantly gives him 1 live and he must collect 3 green gems to get one live.
The gems collected, number of lives left and time ellapsed are shown on top of the screen.</p>
<p>The game has 3 level of difficulties:</p>
<ul>
<li>Level 1: easy, enemies are few and moving slowly.</li>
<li>Level 2: medium, more enemies moving more quickly.</li>
<li>Level 3: many enemies, moving fast and in both directions.</li>
</ul>
<p>Game commands are entered with the keyboard:</p>
<ul>
<li>Before the game starts:
<ul>
<li><code>space</code> starts the game.</li>
<li><code>+</code> slelects the level (cycling choice).</li>
<li><code>Enter</code> selects the image of player (cycling choice).</li>
</ul>
</li>
<li>During the game:
<ul>
<li><code>Esc</code> game pause/resume.</li>
<li><code>Up arrow</code> moves player one square up</li>
<li><code>Down arrow</code> moves player one squere down</li>
<li><code>left arrow</code> moves player one square left</li>
<li><code>right arrow</code> moves player one square right</li>
</ul>
</li>
</ul>

<p>Purpose of the project was to use object oriented code. This realisation is organized with 2 main classes of objects:</p>
<ul>
<li>Game: this class holds everything about the game logic.</li>
<li>Character: the items evolving in the game. those items are 3 sub-classes of this one.
<ul>
<li>Enemy: the enemies crossing the game area.</li>
<li>Player: the player.</li>

</ul>
</li>
</ul>
